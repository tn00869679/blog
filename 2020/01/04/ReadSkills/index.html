How To Read The Books

閱讀分成四個層次 :

1. 基礎閱讀

- 只要看得懂字就可以了。

2. 檢視閱讀

(一)略讀 : 
1. 先看書名頁，然後如果有序就先看序。
2. 研究目錄頁。
3. 如果書中附有索引，也要檢閱一下(參考一下被引用頁數的多寡)。
4. 若是包著書衣的新書，讀一下出版者的介紹。
----------------------------------------------------------------------------------------------------
以上足以判斷是否值得繼續讀下去。
5. 開始對模糊印象中-書的目錄，挑幾個看來跟主題息息相關篇章來看(若開頭或結尾有摘要說明，就要仔細閱讀這些說明)。
6. 最後，把書打開來，東翻翻西翻翻，唸個一、兩段，有時候連續讀幾頁，但不要太多。

(二)粗淺的閱讀 : 
1. 頭一次面對一本難讀的書的時候，從頭到尾先讀完一遍，碰到不懂的地方不要停下來查詢或思索。
2. 逗留與倒退的問題靠手指輔助解決。
3. 摘要 : 在閱讀一本書的時候，慢不該慢到不值得，快不該快到有損於滿足與理解。
主動閱讀的基礎 :
1. 整體來說，這本書到底在談些什麼?
2. 作者細部說了什麼，怎麼說的?
3. 這本書說得有道理嗎?是全部有道理，還是部份有道理?
4. 這本書跟你有什麼關係?
如何讓書真正屬於你自己 :
1. 在重點畫底線。
2. 在畫底線處的欄外再加畫一道線，以示強調。
3. 在空白處做星號或其他符號，慎用，用以表示書中十來個最重要的區塊。
4. 在空白處編號，將論點發展出的一連串重要陳述做順序編號。
5. 在空白處記下其他的頁碼，用以表示相同/相反的論點，整合全書。
6. 將關鍵字或句子圈出來，功能同 " 1 " 。
7. 在書頁的空白處做筆記，將某章節的問題(或答案)記下來，也可以將複雜論點做簡述，或記下全書所有主要論點的發展順序。
前/後空白頁 :
- 在書的前空白頁寫出全書大綱、整體架構(不需要全部重點一一列出)。
- 在書的後空白頁做為個人索引頁，將作者的主要觀點依序記下來。
三種做筆記的方法 :
1. 結構筆記,記下全書的架構(作者如何發展他的觀點)，而不是內容，也不是細節。
2. 概念筆記,分析閱讀時，了解作者觀點且生成自己的觀點。
3. 辯證筆記,主題閱讀時，從多本擁有同主題的類似書籍中摘要出來的討論情境筆記。

3. 分析閱讀

(規則一) 一定要知道自己在讀哪一類書籍，而且越早知道越好，最好在開始閱讀之前就先知道。

4. 主題閱讀
